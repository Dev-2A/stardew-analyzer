# 🌾 Stardew Analyzer

스타듀밸리 세이브 파일(.xml)을 업로드하면 농장 현황, 주민 우정도, 스킬 레벨,  
퍼펙션 달성률을 시각화해서 보여주는 웹 서비스입니다.

## ✨ 주요 기능

- 📁 세이브 파일 업로드만으로 즉시 분석 (설치 불필요)
- 🏡 농장 현황 대시보드 (작물, 동물, 자금 등)
- 💛 주민 우정도 레이더 차트
- 🎯 퍼펙션 달성률 트래커
- 🌟 스킬 레벨 시각화

## 🛠 기술 스택

- **Backend**: Python, FastAPI
- **Frontend**: React, Recharts, D3.js
- **Parsing**: Python xml.etree
